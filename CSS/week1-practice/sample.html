演習2：画像とリンクを複数配置

このページは、HTML/CSS初学者向けの演習課題の完成例です。
画像を3枚並べて、Twitterリンクをボタン風にスタイリングしています。

ソーシャルリンク

使用したCSSのポイント

/* 画像のスタイル */
.gallery-image {
  width: 220px;
  height: 165px;
  margin: 10px;
  border: 3px solid #ddd;
}

/* ボタンのスタイル */
.twitter-button {
  background-color: #1DA1F2;
  color: white;
  padding: 12px 30px;
  border-radius: 25px;
}

学習のヒント：
• 画像のサイズは width と height で指定
• margin で要素間の余白を調整
• padding でボタン内部の余白を調整
• :hover で マウスオーバー時の効果を追加
• transition でアニメーション効果を滑らかに
